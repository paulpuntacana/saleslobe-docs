## What you can build

- **CRM sync** — Push hot leads directly into HubSpot, Pipedrive, or Airtable the moment they reply
- **AI-powered workflows** — Get Corty's reply suggestions in any tool via API
- **Automated reporting** — Pull your Friday and Monday briefs into Slack, Notion, or email
- **White-label products** — Embed Corty's AI into your own SaaS with a `sl_partner_` key

## API version

Current version: **v1**. All endpoints are prefixed `/v1/`.
